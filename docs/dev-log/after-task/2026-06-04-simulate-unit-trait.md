## Scope

Additive R helper only. NO TMB / engine / likelihood / formula-grammar
change. Adds the generic `(unit, observation, trait)` sibling of the
domain-specific `simulate_site_trait()`.

## What changed

- `R/simulate-unit-trait.R` (new): `simulate_unit_trait(n_units = 50L,
  n_obs_per_unit = 3L, n_traits = 5L, alpha = NULL, Lambda_B = NULL,
  Lambda_W = NULL, psi_B = NULL, psi_W = NULL, sigma2_eps = 0.5,
  seed = NULL, ...)`. Keeps the between/within loading machinery
  (`Lambda_B`, `Lambda_W`, `psi_B`, `psi_W`, `sigma2_eps`); OMITS the
  phylogenetic (`Cphy`, `sigma2_phy`, `sigma2_sp`) and spatial
  (`spatial_range`, `sigma2_spa`, `coords`) machinery, which stay in
  `simulate_site_trait()`. All-NULL default = fixed-effects-only
  (trait intercepts + row residual). Returns `list(data, truth)`:
  - `data`: long df with columns `unit`, `observation`, `trait`,
    `value`, `unit_observation` (the row id matching gllvmTMB's
    `unique()` within-unit grouping contract — one level per
    unit-observation cell).
  - `truth`: `alpha`, `Lambda_B`, `Lambda_W`, `psi_B`, `psi_W`,
    `sigma2_eps`. `Cphy`/`coords` are not part of the generic DGP and
    are not returned.
- `R/simulate-site-trait.R`: docstring lines 7-14 now point at
  `simulate_unit_trait()` for the generic case; added a `@seealso`
  cross-link. Behaviour unchanged.
- `man/simulate_unit_trait.Rd` (new) + `man/simulate_site_trait.Rd`
  (regenerated by hand: R/roxygen2 unavailable in this environment;
  the `.Rd` was edited to match the roxygen exactly).
- `NAMESPACE`: `export(simulate_unit_trait)` added (alphabetical slot).
- `tests/testthat/test-simulate-unit-trait.R` (new):
  - (a) recovery — a `gllvmTMB(value ~ 0 + trait + latent(0 + trait |
    unit, d = 2) + unique(0 + trait | unit_observation))` fit recovers
    the true between-unit `Lambda_B` shape (Procrustes per-factor
    cor > 0.90) and the off-diagonal Sigma pattern. **Heavy-gated**
    (`skip_on_cran()` + `skip_if_not_heavy()`) because the TMB fit is
    slow; mirrors the RE-09 recovery harness.
  - (b) the `unit_observation` id matches the `unique()` row contract:
    `nlevels == n_units * n_obs_per_unit`, exactly `n_traits` rows per
    level, bijection with the `(unit, observation)` pair. (light)
  - (c) all-NULL default = coherent fixed-effects-only dataset:
    correct row count, column names, factor levels, no NAs, NULL
    structure components, no `Cphy`/`coords`. (light)
  - plus light seed-reproducibility, truth-honouring, and
    wrong-shape-`Lambda_B` error checks.
