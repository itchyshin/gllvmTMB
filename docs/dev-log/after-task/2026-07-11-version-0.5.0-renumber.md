# After-task — First CRAN release renumbered 1.0.0 → 0.5.0 (D-42)

**Date:** 2026-07-11
**Author:** Claude (with Shinichi)
**Scope:** version-number decision + bounded renumber sweep. No code/API/likelihood change.

## Decision

gllvmTMB's first CRAN release is numbered **0.5.0, not 1.0.0** (brain decision
**D-42**, mirroring drmTMB's D-40). 1.0 is reserved for the capability-maturity
milestone (complete surface + full story + committed-stable API).

Reached by convening the standing review perspectives (Rose, Pat, Emmy, Curie,
Darwin) as a grounded panel — **5–0 for 0.5.0, all high confidence**. The panel
read the actual capability docs, not memory. Decisive grounded evidence:

- **Interval coverage FAILED its production gate** (design 35): CI-08 promoted no
  production RDS — 13/15 cells below the 94% target, only Gaussian d=1/d=3 cleared;
  CI-10 mixed-family coverage down to 0.55. Intervals are half of what a GLLVM user
  does; they are not calibrated.
- **delta/hurdle latent-scale correlation is flagged "Do not advertise"** (design
  61, FAM-17/MIX-10).
- **API still in motion:** the taught keyword grid was reshaped 4×5→4×4 with
  user-visible label changes days before tagging, atop 24 live deprecation call
  sites — not a frozen 1.0 API.
- **Precedent + consistency:** D-40 reserves 1.0 as the maturity milestone and says
  "a 1.0 on a first CRAN entry over-claims"; D-41 says "a 1.0 tag is not maturity."
  gllvmTMB's post-1.0 backlog (Julia parity, the paper, the full coverage campaign)
  fails the full-story prong. A 1.0.0 gllvmTMB beside a 0.5.0 drmTMB is inconsistent.
- **The package's own signals already read 0.x:** README lifecycle badge
  `experimental`, README "preview 0.2.0 pre-CRAN", inst/CITATION 0.2.0.

Strongest surviving counter (recorded, not adopted): under strict SemVer, 1.0
promises a stable *API*, not feature-completeness, and a release advertised strictly
to the validated core (point estimation + phylo/spatial recovery, intervals framed
recovery-only) could carry 1.0 — but that requires surface-narrowing work in
NEWS/README that has not been done.
